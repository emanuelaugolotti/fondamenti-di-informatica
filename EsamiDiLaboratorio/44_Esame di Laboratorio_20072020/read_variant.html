read_variant Submit Solution

Creare i file variant.h e variant.c che consentano di utilizzare la funzione:

  1. extern void *read_variant(FILE *f, char *type);

La funzione read_variant accetta come parametro un puntatore a file f, già aperto in lettura in modalità non tradotta (binario), e un puntatore a char type, e restituisce un puntatore a void.

È definito un formato di salvataggio dati che consente di memorizzare tipi diversi in binario. Il formato parte con un byte tra i seguenti: 

  1. 'c' : char
  2. 's' : short
  3. 'i' : int
  4. 'l' : long long
  5. 'f' : float
  6. 'd' : double

e prosegue con una variabile del tipo corrispondente, in little endian. La funzione legge dal file un byte, e lo mette in type, poi alloca tanta memoria quanta richiesta dal tipo di dato trovato e legge tanti byte quanti richiesti dal tipo nel buffer appena allocato, che poi ritorna.

Se la funzione non riesce a leggere il primo byte, o i dati successivi, o se il carattere non è uno di quelli indicati precedentemente, ritorna NULL.

Nota: la funzione verrà chiamata più volte su file che contengono numerosi dati, quindi non deve aprire o chiudere il file, spostare il puntatore o tornare all’inizio. Dovete limitarvi a leggere i byte necessari lasciando che il cursore interno del file avanzi opportunamente.
